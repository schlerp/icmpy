# Highlights

Identify wins, blockers, and risks.

## Inputs

- Layer 4 (working): `output/raw_metrics.md`
- Layer 3 (reference): `_config/voice.md`

## Process

1. Read the metrics.
2. Extract 2-4 wins and 1-3 risks/blockers.
3. Write one sentence of context for each.

## Outputs

- `output/highlights.md` — wins and risks.
