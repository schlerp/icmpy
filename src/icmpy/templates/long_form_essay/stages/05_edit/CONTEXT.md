# Edit

Polish the draft into a final essay.

## Inputs

- Layer 4 (working): `output/draft.md`
- Layer 3 (reference): `_config/voice.md`

## Process

1. Cut weak paragraphs and tighten transitions.
2. Check that the thesis is supported throughout.
3. Add a brief summary of the argument at the end.

## Outputs

- `output/final.md` — edited essay.
