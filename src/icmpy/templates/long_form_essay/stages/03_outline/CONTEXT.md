# Outline

Create a section-by-section plan for the essay.

## Inputs

- Layer 4 (working): `output/research_notes.md`
- Layer 3 (reference): `_config/voice.md`

## Process

1. Identify 3-5 sections needed to support the thesis.
2. List the point of each section and the evidence it uses.
3. Note transitions between sections.

## Outputs

- `output/outline.md` — section plan.
