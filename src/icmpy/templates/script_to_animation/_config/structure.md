# Structure

## Script sections

- Hook (0-5s): grab attention
- Setup (5-20s): introduce the topic and why it matters
- Body (20-60s): 2-3 key points with evidence
- Payoff (60-75s): synthesis or takeaway
- CTA (75-{{ target_duration }}s): clear next step

## Formatting

- Use Markdown headings for each section
- Keep paragraphs short (1-2 sentences)
- Each section should be readable in roughly 10-15 seconds
